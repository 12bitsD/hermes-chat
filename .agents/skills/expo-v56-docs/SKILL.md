# Expo SDK 56 Project Rule

Before writing project code, read the exact versioned Expo docs at:

https://docs.expo.dev/versions/v56.0.0/

Use the v56 page index to open the specific package or configuration page relevant to the edit. Do not rely on older Expo SDK behavior when changing Expo APIs, app config, permissions, native module usage, or package versions.

Project baseline:

- Expo SDK 56
- React Native 0.85
- React 19.2.3
- Minimum Node.js 22.13.x per Expo SDK 56 docs

This Codex skill is the project-level replacement for the Claude setting that enabled `expo@claude-plugins-official`; no local Claude plugin tree was available to copy.
